The plugin is looking for pictures in the public domain on the Internet.

== Description ==

The plugin allows you to search and use on the site pictures hosted on image hosting sites on the Internet:
Pexels, Pixabay, Unsplash, Flaticon(svg), Iconfinder

Also using the plugin you can upload images directly in the Elementor editor window


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
